# VELOCITES



## <VELOCITES
Receive atomic velocities

**Datatype:** `MDI_DOUBLE`  
**Quantity**: `3 * NATOMS`  

 Engine sends the velocities of atoms in its system to the driver.

## >VELOCITES
Send atomic velocities

**Datatype:** `MDI_DOUBLE`  
**Quantity**: `3 * NATOMS`  

 Driver sends set of atomic velocities to engine.
