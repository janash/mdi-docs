# Simulation Control and Node Management

Commands that control the flow of the simulation and manage the engine's nodes.

Command Name | Description
------------ | -----------
[@](@.md) | Receive current node name.
[@INIT_MC](@INIT_MC.md) | Go to the `INIT_MC` node. Initialize Monte Carlo simulation.
[@INIT_MD](@INIT_MD.md) | Go to the `INIT_MD` node.
[@INIT_OPTG](@INIT_OPTG.md) | Initialize geometry optimization.
[@PRE-FORCES](@PRE-FORCES.md) | Go to pre-forces node
